WELCOME TO XilfteN (pronounced zilfthin)!


To use app, enter a table name in the upper text box where it says to enter table name, the value here will be the
table that is affected when the buttons create table, populate table and delete table are clicked. 

To populate and create table it is vital to enter all of the attribute information in the large text field on the
left of the app, in correct sql format, with the exception of adding semi colons at the end of the file, do not add
a semi colon at the end of the text. 

The same stand true for querying the database, in the enter a query text field enter a query in correct sql format
and click the "Query the database" button, the app will output the results one at a time.
